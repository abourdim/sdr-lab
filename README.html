📡 SDR Kids Lab v1.0.0

Discover the invisible world of radio waves!

A free, interactive, offline-first educational web app teaching Software Defined Radio to kids and beginners. Follow Mouhammed the Radio Explorer through 15 modules from basic radio concepts to building your own SDR station.

🚀 Quick Start

  1. Unzip and open index.html in any modern browser
  2. Click 🚀 Start Adventure
  3. Switch levels (🟢 Newb / 🟡 Explorer / 🔴 Developer) anytime!

✨ Features

  • 45 unique content experiences (15 modules × 3 difficulty levels)
  • 90 quiz questions with instant feedback
  • 45 hands-on challenges scaled by difficulty
  • 75+ fun facts and "Did You Know?" discoveries
  • 10 interactive simulators (Spectrum, Waterfall, Modulation, Antenna, Range)
  • XP & leveling system (10 levels, from Radio Rookie to RF Legend)
  • 15 earnable badges for achievements
  • Printable certificates of achievement
  • Teacher/Parent dashboard with progress reports
  • Full-text search across all content
  • Sound effects (Web Audio API)
  • 🌙 Dark / ☀️ Light themes with Islamic geometric design
  • Accessibility (high contrast, dyslexia font, reduced motion, keyboard nav)
  • PWA offline support — works without internet
  • 100% local — zero tracking, zero data collection
  • 152 KB total — no frameworks, no dependencies
