👨‍🏫 Teacher Guide

How to Use in the Classroom

Select Teacher role in Settings to unlock all modules including HackRF One.

Suggested Lesson Plans

30 min: Module 0 (Welcome) + Module 2 (What is Radio?) — great for introduction class
60 min: Modules 0-3 — full theory foundation
90 min: Modules 0-3 + Module 7 (micro:bit) — theory + hands-on
Full day: Modules 0-9 — complete kids track
Week-long: All 15 modules — full curriculum

Hardware Recommendations

  • Minimum: Just computers — simulation mode works without hardware
  • Basic: Pairs of micro:bits for hands-on radio
  • Standard: micro:bits + RTL-SDR dongles
  • Advanced: All above + ESP32+LoRa + HackRF One (teacher demo)

Safety Notes

  • RTL-SDR is safe for all students (receive-only)
  • HackRF One should only be used by the teacher for demonstrations
  • Always discuss legal and ethical radio use before hands-on activities
  • The app includes built-in safety warnings in Module 13
