🔌 ESP32 + LoRa Guide

LoRa = Long Range radio. It can send messages across a whole city!

What is LoRa?

LoRa is a radio technology designed for IoT (Internet of Things). It sends small amounts of data over very long distances using very little power.

Frequency Bands by Region

RegionFrequency
Europe868 MHz
North America915 MHz
Asia433 MHz

Range

Typical range: 1-10+ km depending on spreading factor, antenna, and obstacles.

What You Need

  • Two ESP32 boards with LoRa modules (e.g., Heltec WiFi LoRa 32)
  • USB cables
  • Arduino IDE or PlatformIO
  • Antennas (usually included)
